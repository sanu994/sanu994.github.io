Sample API Documentation

GET /api/users

Get a list of users

Parameters

limit (query): Number of users to return (default: 10)

Responses

200 OK 
[
  {
    "id": 1,
    "name": "John Doe",
    "email": "john@example.com"
  },
  ...
]
POST /api/users

Create a new user

Request Body

{
  "name": "string",
  "email": "string",
  "password": "string"
}

Responses

201 Created 
{
  "id": 1,
  "name": "John Doe",
  "email": "john@example.com"
}
PUT /api/users/{id}

Update an existing user

Parameters

id (path): User ID

Request Body

{
  "name": "string",
  "email": "string"
}

Responses

200 OK 
{
  "id": 1,
  "name": "Updated Name",
  "email": "updated@example.com"
}
DELETE /api/users/{id}

Delete a user

Parameters

id (path): User ID

Responses

204 No Content
